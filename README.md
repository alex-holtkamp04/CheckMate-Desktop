# CheckMate-Desktop
C# 4400 Final Project

# Usage Instructions:
__Important Note:__ This application uses .NET 10.0

### Downloading the application:
Download the provided zip file containing the solution file along with the executable

### Running the application:
If using an executable file:
- Within the unzipped file provided, go to the `CheckmateDesktop\bin\Debug\net10.0-windows` folder and run the .exe file
- Run the Executable file provided
- To stop, simply close the application

Or, if not using the executable file:
- Open the solution file using Visual Studio(Recommended: 2026 or other version that uses .NET 10.0)
- Build and run the solution
- To stop, simply close the application

### How to use:
- Click a chess piece to select it
- The highlighted destinations will display valid moves
- Click a  highlighted destination square to move the piece
- Turns automatically change between white and black, and are displayed in the top left of the active player
- The game continues until a checkmate/stalemate condition is reached
